# 💡Calculadora de Consumo Elétrico Inteligente
Este é um programa em Python desenvolvido para ajudar usuários a estimarem o consumo mensal de energia de aparelhos eletrodomésticos e o custo aproximado na conta de luz.

## 📍Objetivo
O sistema recebe dados técnicos simples e transforma em informações úteis para o controle de gastos domésticos.

## 🧮Fórmula Utilizada
O cálculo segue a lógica:
`consumoMensal = (potencia * horasDia * 30) / 1000`

O projeto também calcula o valor em Reais baseando-se em uma tarifa fixa de R$ 0,75 por kWh.

## 💻Como executar
1. Certifique-se de ter o Python instalado.
2. Clone este repositório.
3. Execute no terminal: `python app.py`.
